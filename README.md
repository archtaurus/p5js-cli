# The CLI for p5.js

[![Build and Publish](https://github.com/archtaurus/p5js-cli/actions/workflows/publish.yml/badge.svg)](https://github.com/archtaurus/p5js-cli/actions/workflows/publish.yml)

## Install

``` shell
npm i -g p5js
```

## Usage

``` shell
p5js new <sketch>
cd <sketch>
code . && p5js run
```
